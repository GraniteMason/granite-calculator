# 🏛️ Мемоліум Київ - Памятники та Гранітні вироби

Професійний веб-сайт для замовлення памятників, гранітних виробів та благоустрою на кладовищах Київу.

**Відкрита на:** http://localhost:8000

## 🎯 Основні функції

### 📸 Портфоліо
- Галерея завершених проектів
- Приклади паматників, стел, хрестів та меморіалів
- Інформація про кладовище та рік виготовлення

### ⚰️ Типи паматників
- **Плита** - класичний вибір
- **Стела** - з портретом та письмом
- **Хрест** - символ вічності
- **Меморіал** - люкс проекти

### 🗺️ Кладовища Киева
- Інформація про 6+ головних кладовищ міста
- Адреса та описи кожного
- Район розташування

### 📋 Послуги
- Виготовлення паматників
- Монтаж та установка
- Гранітні вироби (стільниці, підвіконники)
- Премієм оздоблення
- Доставка
- Ремонт та реставрація

### 💰 Калькулятори
**Паматник**
- Вибір матеріалу (граніт, мрамур, преміум)
- Введення розмірів
- Автоматичний розрахунок цены

**Ограда на гробницю**
- Тип (металева, кована)
- Розміри ділянки
- Розрахунок по периметру

**Лампадки**
- Вибір количества (0, 2, 4, 6)
- Фіксована ціна

**Гранітні вироби**
- Тип виробу
- Матеріал
- Розміри
- Автоматичний розрахунок вартості

### 📱 Адаптивний дизайн
- Оптимізовано для мобільних пристроїв
- Планшети та десктопи
- Гладка прокрутка та анімація

## 🛠️ Технологи

- **HTML5** - Семантична структура
- **CSS3** - Сучасний дизайн з градієнтами
- **JavaScript (ванільний)** - Розрахунки в реальному часі
- **JSON** - Конфігурація даних

## 🚀 Запуск

**Локально:**
```bash
python3 -m http.server 8000
# Відкрийте http://localhost:8000
```

**На сервері:**
Просто завантажте всі файли на ваш веб-сервер

**GitHub Pages:**
```bash
git push origin main
```

_После первого пуша настроенный workflow автоматически будет публиковать сайт на GitHub Pages._

Ожидаемый URL: `https://GraniteMason.github.io/granite-calculator/`

## 💾 Конфігурація

Всі дані в `js/config.json`:

```json
{
  "cemeteries": [...],          // Кладовища
  "monumentTypes": [...],       // Типи паматників
  "portfolio": [...],           // Портфоліо проектів
  "graniteTypes": {...},        // Типи гранітуу
  "monuments": {...}            // Матеріали, огради, лампадки
}
```
