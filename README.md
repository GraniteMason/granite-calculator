# Granite Calculator

## Project Documentation

This is the documentation for the Granite Calculator project. It provides an overview of the project's purpose, installation instructions, usage, and other relevant information.

### Purpose
The Granite Calculator is designed to help users with calculations related to granite materials, including area calculations, volume calculations, and cost estimations.

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/GraniteMason/granite-calculator.git
   ```
2. Navigate to the project directory:
   ```
   cd granite-calculator
   ```
3. Install dependencies (if any).

### Usage
To use the Granite Calculator, run the main application script:
```bash
python main.py
```
